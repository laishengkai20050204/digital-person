# 自然状态部署验证

第五批部署后按真实时间连续观察，不通过手工修改状态制造结论。

重点验证：

1. 四小时睡眠只产生部分恢复，八小时睡眠恢复更多；
2. 深夜、长时间清醒和睡眠债会提高困倦；
3. 进食期间饥饿下降，进食结束后普通空腹效果不会继续叠加；
4. 学习或工作结束后，专注、动机和认知负荷逐渐回归基础水平；
5. 同一事件重试时，强度档位映射得到相同的数值；
6. `INSTANT` 仅用于十分钟以内的突发固定时间效果；
7. 活动决策上下文能够看到活动持续时长等级、最近睡眠量、睡眠债和距上次进食时间。

事件效果生命周期测试使用不受自然生理层影响的状态维度，避免把进食、睡眠等确定性自然变化误判为事件效果越界。测试期望值同时按照各维度真实边界计算，例如 `VALENCE` 的负向变化趋近 `-1`，而不是趋近 `0`。

当前不设置“睡眠不足六小时禁止醒来”的硬规则。只有自然状态和决策提示仍无法阻止无理由短睡眠醒来时，才增加 Java 侧约束。
